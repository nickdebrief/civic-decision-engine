# Research Publications

This directory contains the published research materials for the Civic Record Evaluation Framework (CREF).

## Publications

### Version 1.0
The foundational paper establishing CREF as a deterministic methodology for evaluating what visible records can support through structured classifications, administrative pathways, traceability mechanisms, and reproducible outputs.

### Version 2.0
Extends Version 1.0 by introducing methodological provenance, evidence attribution, explainability certification, framework self-description, and reflexive consistency.

## Research Principle

CREF asks:

> What can be shown from the visible record?

rather than:

> What happened?

The framework evaluates visible records and existing record-derived outputs without determining truth, liability, intent, wrongdoing, or real-world evidential sufficiency.

## Properties

- Deterministic
- Traceable
- Explainable
- Inspectable
- Reproducible
