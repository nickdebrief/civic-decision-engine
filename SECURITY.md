# Security Policy

## Purpose

The Civic Decision Engine explores transparency, accountability, and behaviour modelling in civic and institutional systems.

Because projects involving decision models and civic infrastructure can raise security or misuse concerns, responsible disclosure of potential issues is encouraged.

---

## Reporting a Security Issue

If you discover a security vulnerability, misuse risk, or integrity concern related to this repository, please report it responsibly.

Please include:

- A clear description of the issue
- Steps to reproduce the problem if applicable
- Any potential impact you believe the issue may have

Reports should aim to improve the reliability and integrity of the project.

---

## Responsible Disclosure

Security issues should be reported privately before public disclosure so they can be reviewed and addressed appropriately.

Please avoid posting security vulnerabilities publicly in issues or discussions.

---

## Scope

Security concerns may include:

- vulnerabilities in code or dependencies
- risks related to misuse of the model
- integrity issues affecting data or audit logs
- behaviour scoring manipulation risks

The goal is to maintain transparency while ensuring the system cannot be easily abused.

---

## Commitment

All security reports will be reviewed and addressed in good faith.

The purpose of this policy is to encourage responsible collaboration in improving the safety and integrity of the project.
